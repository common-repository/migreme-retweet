== Description ==
**Portugu&ecirc;s**
Esse plugin permite ao adminstrador adicionar um bot&atilde;o para que os usu&aacute;rios possar retweetar o artigo com um link do [Migre.me](http://migre.me/) .

Importante<br>
Se voc&ecirc; quiser ajudar no projeto entre em contato comigo no site [Webord](http://webord.net/desenvolvimento/migre-me).

**English**
This plugin allows the Admnistrator to add a button on your blog for your users retweet an article with [Migre.me](http://migre.me/) link.

Important<br>
If you would like to help with this project, contact me on my website, [Webord](http://webord.net/desenvolvimento/migre-me).

== Installation ==

**Portugu&ecirc;s**

1. Extraia o arquivo na pasta `/wp-content/plugins/`. Se voc&ecirc; estiver fazendo o upload, n&atilde;o se esque&ccedil;a de fazer o upload da pasta inteira, n&atilde;o apenas os arquivos `.php`, para  `/wp-content/plugins/`.
1. Ative o plugin na parte de 'Plugins' do WordPress
1. Visite a se&ccedil;&atilde;o de op&ccedil;&otilde;es do Migre.me no seu menu (*Migre.me - Op&ccedil;&otilde;es*), e configure o plugin como desejar
1. Pronto!
	
	
**English**

1. Unzip the file into your `/wp-content/plugins/` directory. If you're uploading it make sure to upload the top-level folder. Don't just upload all the php files and put them in `/wp-content/plugins/`.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit your Migre.me options (*Migre.me - Op&ccedil;&otilde;es*), and set the options
1. That's it!
